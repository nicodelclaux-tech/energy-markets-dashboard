# Visual Design System

## When to Use

- Styling or theming changes.
- Visual refinement of new dashboard modules.
- Aligning charts and UI elements into a single presentation system.

## Procedure

1. Preserve the professional institutional feel of the product.
2. Define hierarchy through typography, spacing, and restrained color use.
3. Support readability for dense market data views.
4. Ensure empty, stale, and warning states remain visually explicit.

## Output

- Cohesive design direction.
- Styling rules for the affected feature.
